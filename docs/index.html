The problem

Raw analogRead() is never enough.

You've been there — ADC noise produces phantom readings, the sensor drifts after 30 minutes, and recalibration means reflashing the firmware every single time.

K

Adaptive Kalman Filter

No more jittering at 0A. The filter locks onto stable readings and only reacts when a real change happens — not when the ADC spits noise from your motor driver.

C

Multi-Point Calibration

Your sensor isn't linear. We know. Add up to 10 reference points at real voltages — the library interpolates the rest. No more hardcoded magic numbers.

S

Persistent Storage

Calibrated on-site? It stays. EEPROM on AVR, NVS on ESP32 — all automatic. Power cycle, firmware update, doesn't matter. Your calibration survives.

Z

Auto-Zero Compensation

After 30 minutes your ACS712 offset drifts from heat. IndustrialSense notices when you're at 0A for 5+ seconds and silently recalibrates itself.

64

64× Oversampling

One ADC sample has ±2 LSB noise. We average 64. That's 3 extra bits of effective resolution — 13-bit accuracy from a 10-bit ADC. In under 1ms.

fn

Custom ADC Injection

Outgrew the built-in ADC? Pass a function pointer to your ADS1115 or MCP3008 read function. The entire pipeline keeps working — no library fork needed.

Quick Start

Three lines. That's it.

No config structs. No init sequences. No 200-line boilerplate. Declare, begin, read.

01_Basic_Usage.ino 
#include <IndustrialSense.h>

// Declare sensors — library auto-loads all defaults
IndustrialCurrent currentSensor(ACS712_20A, A0);
IndustrialVoltage voltageSensor(A1, 30000.0, 7500.0);

void setup() {
    currentSensor.begin();
    voltageSensor.begin();
}

void loop() {
    float amps  = currentSensor.read();  // Filtered current
    float volts = voltageSensor.read();  // Smoothed voltage
    delay(500);
}
Under the hood, each read() call runs the full pipeline: oversample → filter → convert → deadzone → smooth. You get lab-grade readings from a $2 sensor.

Runtime Calibration

Calibrate without reflashing.

Deployed in the field? Connect via Serial, MQTT, or Bluetooth and calibrate on the spot. Changes persist across reboots.

v12.5 Read current ADC value and map it to 12.5V. Saves to persistent storage automatically. Supports up to 10 calibration points.
v0 Calibrate the zero-voltage reference point. Connect the input to GND, then send this command.
auto Force the current sensor to read the zero-offset immediately. Compensates for thermal drift. Saves to EEPROM/NVS.
reset Clear all calibration data from both RAM and persistent storage. Sensor returns to factory defaults.
Calibration Workflow 
// In your loop(), add this to enable serial commands:
if (Serial.available()) {
    String cmd = Serial.readStringUntil('\n');
    voltageSensor.processCommand(cmd);  // Handles "v12.5", "reset"
    currentSensor.processCommand(cmd);  // Handles "auto", "reset"
}
Reference

API.

Complete method reference for both sensor classes.

IndustrialCurrent
void begin()
Initialize the sensor. Auto-detects ADC resolution, loads saved offset from persistent storage, seeds the Kalman filter.
float read()
Full pipeline: oversample → Kalman filter → convert to Amps → deadzone → auto-zero check. Returns filtered current in Amperes.
float readRaw()
Returns raw oversampled ADC value (before Kalman). Useful for diagnostics.
void setKalmanProcessNoise(float Q) Advanced
Set process noise Q. Higher → trust measurements more (noisier output). Default: 0.01
void setKalmanMeasurementNoise(float R) Advanced
Set measurement noise R. Higher → trust model more (smoother but slower). Default: 0.5
void setKalmanInnovationThreshold(float rho) Advanced
Innovation threshold for adaptive skip. If |change| < rho, skip full Kalman update. Set 0 to disable. Default: 0.5
void setOversampleCount(uint16_t count)
Number of ADC samples to average per read(). More = better resolution, slower. Default: 64
void setDeadzone(float amps)
Readings below this threshold are forced to 0.0A. Default: 0.20A
void forceAutoZero()
Immediately recalibrate the zero offset using the current ADC reading.
void setCustomADCFunction(CustomADCReadFunc func) Advanced
Inject a custom ADC read function. Signature: int myRead(uint8_t pin). For external ADCs like ADS1115.
void saveCalibration() Persistence
Save current zero-offset to EEPROM/Preferences.
bool processCommand(const String& cmd)
Parse and execute a calibration command string. Supports "auto" and "reset". Returns true if recognized.
IndustrialVoltage
void begin()
Initialize the sensor. Loads calibration map, seeds EMA with initial reading.
float read()
Full pipeline: oversample → interpolation (or divider math) → EMA smooth. Returns voltage in Volts.
bool addCalibrationPoint(float realVoltage)
Auto-reads current ADC and maps it to the given voltage. Array is auto-sorted. Returns false if map is full (10 points).
void setEmaAlpha(float alpha)
Set EMA smoothing factor. Range (0, 1]. Higher = less smoothing. Default: 0.15
void saveCalibration() Persistence
Save all calibration points to EEPROM/Preferences.
bool processCommand(const String& cmd)
Parse a calibration command. Supports "v<value>" and "reset". Returns true if recognized.
